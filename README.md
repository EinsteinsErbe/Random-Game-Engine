# Random Game

## Getting Started

### Prerequisites

Java 8

runs on Windows, Linux and Mac

### Installing

No installing needed!

Just download the most recent [JAR](https://github.com/Major94/Random-Game-Engine/releases) and run it.

* [Gradle](https://gradle.org/) - Dependency Management
